la fonction aParle nous indique si A a parler a B (1 pour oui et 0 pour non) 
dans la foncion "trouver cible" on utilise une pile pour stacker les personnes,
ensuite on compare les 2 derniers de la pile jusqu'a que la pile ait une seule personne ( potentiellement la cible ) 
a la fin on verifie si la cible potentielle est connue de tous et qu'elle ne connait personne

Pour utiliser le programme, il faut dans le fichier "matrice.in" ecrire le nombre de personne et d'ecrire une matrice correspondante au probleme
